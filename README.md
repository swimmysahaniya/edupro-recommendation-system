# 🎓 EduPro Personalized Course Recommendation System

An end-to-end Machine Learning project that segments students and provides personalized course recommendations.

## 🚀 Features
- User segmentation using KMeans clustering
- Feature engineering (Diversity Score, Learning Depth, Enrollment Frequency)
- Recommendation system based on cluster behavior
- Interactive dashboard using Streamlit

## 📊 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
